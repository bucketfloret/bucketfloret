# 💐 Bucket

Bucket is an AI platform that brings together multiple intelligent tools in one place.

## Vision

Instead of using dozens of separate AI services, users can build their own "bouquet" of AI assistants inside Bucket.

## Planned features

- 🤖 AI Chat
- 💻 Coding Assistant
- 📄 PDF Analysis
- 🌍 Translation
- 🎨 Image Generation
- 🧠 Long-term Memory
- 🔍 Web Search

🚧 Early development.
